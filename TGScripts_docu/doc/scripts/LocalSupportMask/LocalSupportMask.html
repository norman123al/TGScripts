1 Description

[hide]

The deconvolution process in pixinsight offers the possiblity to provide a so-called local support mask which is essentially a star mask to protect (bright) stars from deconvolution artifacts. This script automates the generation of a local support mask and provides the essential parameters to influence the mask generation. The script works on linear grayscale or color image.

The image is required to be linear because deconvolution is applied before the image is stretched. A mask is best generated from the luminance data, thus the source image should be ideally grayscale (e.g. luminance extracted from RGB image). The preview control shows either the MLT processed intermediate stage or the overlay preview of the mask (see Preview section below).

2 Parameter Reference

[hide]

Target Image

The target image control allows to choose a main view of a linear image (either grayscale or RGB) as a target for local support mask generation.

Strength

The 'Strength' parameter specifies the brightness of the star mask and thus the amount of protection of the stars.

Scale

The 'Scale' parameter specifies the number of wavelet layers for star mask generation.

Large Scale

The 'Large-scale' parameter determines the growth of large scale structures in the star mask.

Small Scale

The 'Small-scale' parameter determines the growth of small scale structures in the star mask.

Compensation

The 'Compensation' parameter specifies the number of wavelet layers small scale growth compensation.

Smoothness

The 'Smoothness' parameter determines the smoothness of the mask structures.
